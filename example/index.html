# The aggmap example on Wisconsin Diagnostic Breast Cancer (WDBC) dataset : The WDBC is a widely used machine learning benchmark dataset for feature selection methods, which obtained by the university of Wisconsin Hospital to classify tumors as benign or malignant. It contains 30 real-valued features of cell nucleus and 569 samples that are labeled as 357 benign status and 212 malignant status. On breast cancer detection task, the sensitivity and specificity for AggMapNet is 98.0% and 95.5 % Dua, D. & Graff, C. UCI machine learning repository, Wisconsin Diagnostic Breast Cancer (WDBC) Data Set. URL http://archive.ics.uci.edu/ml 37 (2019).
